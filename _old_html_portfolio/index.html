Projects

Expert-Call RAG Assistant

This project is a locally-run Retrieval-Augmented Generation (RAG) system that allows users to query a collection of expert-call transcripts. The system uses a large language model to generate answers based on the most relevant retrieved text chunks and provides citations to the original source document and page number. This ensures that the answers are grounded in the provided context, reducing hallucinations and improving trust in the output. The project is designed to be modular, allowing for easy replacement of components such as the language model or the retrieval system. The primary technologies used were Python, with libraries such as Gradio for the web interface, LanceDB for the vector store, and LlamaIndex for orchestrating the RAG pipeline. The system uses a quantized Llama-3.1-8B model for local inference, Sentence-Transformers for embedding and re-ranking, and PyMuPDF for PDF text extraction.

Scalable Vector Search for Music Recommendations

This project involved building a large-scale, embedding-based recommendation system for music using the Spotify Million Playlist Dataset. We generated vector embeddings for songs, playlists, and lyrics using Word2Vec, stored them in MongoDB and ChromaDB, and used cosine similarity and k-NN for similarity search. Apache Spark powered scalable data processing, and PCA-based visualization revealed meaningful song clusters, enhancing recommendation quality.
